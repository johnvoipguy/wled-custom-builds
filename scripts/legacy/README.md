# Legacy scripts

These scripts are preserved as reference material for the deprecated branch-per-version and clone-per-version workflow.

Do not use them for new work on `main`. Use `../apply-target.sh` and `../build-target.sh` with the `targets/` layout instead.

Legacy-reference entries currently include:
- `quick_setup_wled.sh`
- `quick_setup_wled-seeed-xiao.sh`
- `upgrade_wled_version.sh`
- `wled_manager.sh`
- `setup_esp32s3_w5500.sh` (historical Waveshare/W5500 migration helper; not the canonical target workflow)
